ops_ts.hump_decay(arr, p, relative, axis=1)#

This operator helps to ignore the values that changed too little corresponding to previous ones. It can be used in two modes.

The default mode:

if relative == False:

if abs(x[t] - x[t-1])> p, return x[t], else x[t-1]

The relative mode:

if relative == True:

if abs(x[t] - x[t-1])> p * abs(x[t] + x[t-1]), return x[t], else x[t-1]

Parameters:

  • arr (np.ndarray) – The input array. Must be 2D.

  • p (float) – The threshold value.

  • relative (int) – The mode of the operator. 0 for default mode, 1 for relative mode.

  • axis (int, optional) – The axis along which to calculate the operator. Default is 1.

Returns:

same shape as input array.

Return type:

np.ndarray

ops_ts.jump_decay(arr, sensitivity, force, window, stddev=0, axis=1)#

jump_decay(x) = abs(x-ts_delay(x, 1)) > sensitivity * ts_stddev(x,d) ? ts_delay(x,1) + ts_delta(x, 1) * force: x

If stddev enabled, jump threshold will be calculated as sensitivity * stddev otherwise it is sensitivity

Parameters:

  • arr (np.ndarray) – The input array. Must be 2D.

  • sensitivity (float) – The threshold value.

  • force (float) – The force value.

  • window (int, optional) – The window size. Default is None.

  • stddev (int, optional) – The mode of the operator. 0 for default mode, 1 for relative mode.

  • axis (int, optional) – The axis along which to calculate the operator. Default is 1.

Returns:

same shape as input array.

Return type:

np.ndarray
