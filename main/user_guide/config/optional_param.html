Optional Parameters#

prep_cfg#

数据预处理

Support Method:

filter_event

filter event in horizon

Parameters
start_eventstr

event start

end_eventstr

event end

Example:

prep_cfg = {
    "filter_event": {
        "start_event": "start",
        "end_event": "end"
    }
}
comp_cfg#

算子设置

Parameters:

ops_setlist {‘arith_ops’, ‘trans_ops’, ‘cross_ops’, ‘ts_ops’}

The set of operators to be used

exclude_opslist

The operators to be excluded

Example:

comp_cfg = {
    'ops_set': ['arith_ops', 'trans_ops', 'cross_ops', 'ts_ops'],
    'exclude_ops': ['ts_median', 'ts_percentage']
}
